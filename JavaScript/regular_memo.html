i 는 맨 앞의 하나를 출력 g 는 찾은 모든것을 출력 i(ignore) : 대소문자 구분안함 g(global) : 검색에 맞는(일치되는) 모든 것을 찾겠다. (match가? 배열 형태로 나열해줌) ex_ s[1],s[2],s[3] m : 줄바꿈을 모두 포함해서 찾는 의미 [] : 대괄호 안에 열거된 문자 한개씩 검색하겠다. ^ : 해당하는 것만 빼고 나머지를 찾겠다 [abc] 대괄호 안에 열거된 문자는 [a], [b], [c]로 하나씩 나눠서 검색한다. /abc/gi 는 단어를 찾겠다(대괄호가 없으면 단어 없으면 문자(글자?)) /[A-e]/g; = A~Z, a-e 까지 찾겠다(아스키코드상 대문자가 코드가 더 작으니까) /(red|green|blue)/g; : red와 green과 blue를 찾겠다.(추가 가능) new : 연산자 ex_ Date() : 생성자함수 아래의 두 코드는 같다. new RegExp("(x|y)", "g") or /(x|y)/g /h.t/g; : h로 시작해 t로 끝나는 단어를 찾는다. /\w/g; : 문자 숫자를 찾는다.(63개의 문자를 카리킴) /\W/g; : 63개 문자를 제외하고 찾겠다 /\d/g; : 숫자만 찾겠다. /\D/g; ; 숫자 제외하고 찾겠다. /\s/g; : 공백만 찾겠다. search 값은 숫자 "Hello Look at you" /LO\b/i 의 결과는 3 /\x57/g; : 아스키코드 x57에 해당하는 문자를 표현 /o+/g; : 앞에 쓴 문자의 빈도 수(몇 번 표현해도 되느냐?) 문자 o를 최소 1개이상 표현된걸 찾는다 결과는 ooo,o,o,oo 이런식 /o*/g; o를 생략 또는 1개이상 표현 /lo*/g; l, lo, loo 가능한 것 /10?/g; : 뒤에 뭐가 있어도 10이 존재하면 뽑아온다. 1만 존재해도 뽑아온다. /\d{숫자}/g; : d{4}이면 네자릿수 이상을 뽑아옴 d{3,4}로 뽑아올 수도 있다. d{3.};으로 전부 다 뽑아올 수 있다. /^[가-힣]{2,10}$/; : /^ 제시한 내용으로 시작해서 ~로 끝나야한다 $/
